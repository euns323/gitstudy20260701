## github에 코딩 파일 업로드
1. 내 프로필 > Repositories 클릭 
2. [NEW] 버튼 클릭
3. Repository name에는 업로드할 코딩파일 제목 작성
4. Create Repositories 클릭
5. VS Code 좌측 패널에서 git 아이콘 클릭
6. Initialize Respository 클릭
7. Changes에 추가한 파일 목록 나옴
7-1. git에 업로드 할 파일만 [+]눌러서 Staged Changes로 이동 
8. Message 입력칸에 어떤 것이 변경되었는 입력하고 commit 버튼 클릭
9. 터미널에 아래 코드 순차적으로 붙여넣기
git remote add origin https://github.com/euns323/gitstudy20260701.git
git push -u origin main
